# 화면 디자인

예시 디자인을 그대로 따르지 않고, 필요한 기능이 빠지지 않도록 화면 구조만 정리한다. 실제 스타일은 구현 시 결정한다.

## 공통 네비게이션
- 로고: Smart Blog
- 메뉴: My Blog / Saved Posts / Settings
- 우측: 프로필 아이콘

## 화면 1 — Saved Posts
- 페이지 타이틀 + 한 줄 설명
- 우측 상단: "블로그 생성" 버튼 → Create Blog로 이동
- 카드 그리드 (한 행 3개 기준)
  - 브랜치 태그 + 작성 날짜
  - 제목
  - 썸네일 (선택)
  - 미리보기 텍스트 (말줄임)
  - "수정하기" / "발행하기" 버튼
- 마지막 카드: "새 초안 작성" (점선 placeholder, 클릭 시 Create Blog로 이동)

## 화면 2 — Create Blog
- Repository 선택 (드롭다운)
- Branch 선택 (드롭다운)
- Commit 다중 선택 (체크박스 목록 + 메시지·날짜 표시)
- "초안 생성" 버튼 → 로딩 → 결과 표시
- 결과 영역: AI가 생성한 markdown 초안
- 편집기 (textarea + 미리보기)
- "저장" 버튼

## 화면 3 — 편집 (저장된 포스트의 "수정하기")
- Create Blog의 편집기 영역만 단독으로 표시
- "저장" / "취소" / "발행하기" 버튼
