# PRD - Smart Blog (GitHub 연동 블로그 서비스)

## 1. 목적
GitHub 커밋/코드 변경사항을 AI가 분석해서
개발 블로그 초안을 자동 생성하는 서비스

## 2. 맥락
- 개발자는 잔디는 열심히 심지만 블로그는 안 씀
- 커밋 내역이 곧 개발 기록인데 이걸 글로 변환해주면 유용

## 3. 요구사항
### 블로그 생성
- GitHub Repository 리스트 불러오기
- 브랜치 및 커밋 선택
- 선택한 커밋을 LLM으로 분석 → 블로그 초안 생성
- 생성된 글 편집 가능

### 포스트 관리
- 저장된 포스트 카드형 목록 조회
- 포스트 수정 및 발행

## 4. 검증 기준
- GitHub repo 선택 → 커밋 목록 정상 조회
- 커밋 선택 → AI 블로그 초안 생성
- 생성된 글 편집 후 저장 가능
- 저장된 포스트 목록 조회 가능

## 5. 제약과 예외
- GitHub API 토큰은 서버에서만 사용 (.env)
- OpenAI API 토큰 절대 클라이언트 노출 금지
- 에러 시 사용자에게 명확한 메시지 표시
