# Commit to Blog

GitHub 활동 데이터를 분석해 자동으로 개발 블로그 초안을 생성하는 서비스입니다.

## 1주차 구현 범위

- React 기반 화면 구조 구성
- GitHub Repository 선택 UI
- Branch 선택 UI
- Commit 목록 선택 UI
- AI 블로그 초안 편집기 UI
- 저장된 포스트 카드 목록 UI
- mock 데이터 기반 화면 흐름 연결

## 실행 방법

```bash
npm install
npm run dev
```
