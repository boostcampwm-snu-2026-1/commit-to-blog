## 1. 블로그 섹션

## 2. 검색 필터 바

```
┌────────────────────────────┬──────────────────────────── [🔍]┐
│ 주제                        │ 날짜                             │
│ 주제 검색하기                │ 날짜 추가하기                     │
└────────────────────────────┴──────────────────────────────────┘
```

- 좌측: 주제 필터 영역 / 우측: 날짜 필터 영역 / 맨 우측: 검색 실행 버튼(🔍)
- 세로 구분선(divider)으로 주제·날짜 영역 구분
- 값이 선택된 경우 placeholder 텍스트 대신 선택된 값 표시
0 복수 선택된 주제가 많을 때 첫번쨰 카테고리 이외에는 축약하여 표현 (예: `React 외 2개`)

  | 상태 | 주제 영역 표시 | 날짜 영역 표시 |
  |------|--------------|--------------|
  | 미선택 | `주제 검색하기` | `날짜 추가하기` |
  | 선택됨 | 선택한 주제명(들) | `YYYY.MM.DD ~ YYYY.MM.DD` |

### 주제 필터 동작

1. 주제 영역 클릭 → **주제 모달** 표시 (검색 바 하단에 붙어서 열림)
2. 주제 모달 내 `주제 검색하기` 클릭 → 모달 내 텍스트 input 활성화, 카테고리 실시간 검색 가능
3. 카테고리 **복수 선택** 가능 (체크박스 또는 토글 형태)
4. 선택 완료 후 모달 외부 클릭 또는 닫기 → 모달 닫힘, 선택된 주제명이 필터 바에 표시

#### 주제 모달 구조

```
┌─────────────────────────────────┐
│ [주제 검색하기 input]             │  ← 클릭 시 활성화
├─────────────────────────────────┤
│ ☑ 카테고리 A                     │
│ ☐ 카테고리 B                     │
│ ☐ 카테고리 C                     │
│ ...                             │
└─────────────────────────────────┘
```

- 카테고리 목록은 API에서 동적으로 받아옴
- 백엔드 개발 전에는 `src/mocks/blog-categories.json` 사용
- input에 텍스트 입력 시 카테고리 목록 실시간 필터링

### 날짜 필터 동작

1. 날짜 영역 클릭 → **날짜 모달** 표시 (검색 바 하단에 붙어서 열림)
2. 기간 선택 방식: **시작일 ~ 종료일** 선택
3. 선택 완료 후 모달 외부 클릭 또는 닫기 → 모달 닫힘, 선택 기간이 필터 바에 표시

#### 날짜 모달 구조

- 달력(캘린더) UI로 시작일·종료일 선택
- 시작일 선택 후 종료일 선택하는 2-step 방식
- 시작일~종료일 사이 날짜 하이라이트 표시

### 검색 실행

- 🔍 버튼 클릭 시 선택된 주제·날짜 조건으로 API 요청
- 조건이 없는 필터는 요청에서 제외 (전체 조회)

## 3. 블로그 글 목록

```
┌──────────┐  블로그 제목
│  썸네일   │  YYYY.MM.DD
│  이미지   │  한 줄 소개 텍스트
└──────────┘
```

- 썸네일 이미지 + 제목 + 날짜 + 한 줄 소개로 구성된 카드 형태
- 카드 클릭 시 해당 블로그 글로 외부 링크 이동 (`target="_blank"`)
- 필터 결과가 없는 경우 빈 상태 UI 표시 (문구 미정)

---

## 4. 데이터 / API 연동

### 백엔드 API

- **제공 데이터**:
  - 블로그 카테고리 목록
  - 블로그 글 목록 (주제·날짜 필터 파라미터 지원)
- **요청 파라미터** (필터 없으면 생략, 전체 조회):
  - `categories`: 선택된 카테고리 목록 (복수, 배열)
  - `startDate` / `endDate`: 기간 필터 (`YYYY-MM-DD`)
- **백엔드 개발 전**: 카테고리는 `src/mocks/blog-categories.json`, 글 목록은 `src/mocks/blog-posts.json` 사용

### blog-categories.json 스키마

```json
{
  "categories": ["React", "TypeScript", "알고리즘", "CS"]
}
```

### blog-posts.json 스키마

```json
{
  "posts": [
    {
      "id": "string",
      "title": "블로그 제목",
      "date": "YYYY.MM.DD",
      "description": "한 줄 소개",
      "thumbnail": "/images/blog/thumbnail.png",
      "url": "https://velog.io/@Yeonu-Kim/...",
      "categories": ["React", "TypeScript"]
    }
  ]
}
```
