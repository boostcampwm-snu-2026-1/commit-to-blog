# Review Checklist

> /evaluator 실행 시 로드한다.

## 기능 검증

- [ ] 완료 조건을 모두 충족하는가
- [ ] 엣지 케이스를 처리했는가
- [ ] 에러 상태를 처리했는가

## 코드 품질

- [ ] 불필요한 console.log가 없는가
- [ ] 하드코딩된 값이 없는가
- [ ] 네이밍이 @rules/naming.md 를 따르는가
- [ ] 중복 코드가 없는가

## 보안

- [ ] 사용자 입력을 검증하는가
- [ ] 민감한 정보가 노출되지 않는가
- [ ] XSS / SQL Injection 취약점이 없는가

## 문서

- [ ] 변경된 도메인의 docs/ 파일을 업데이트했는가
- [ ] 새로운 의사결정이 있다면 adr/에 기록했는가
